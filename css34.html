Text Shadow

CSS3 Text Shadow adds shadows to text characters. The Syntax is almost similarly to Box Shadow, except Spread and inset as they are not allowed in Text Shadow.

text-shadow:0px 0px 5px red;

text-shadow:0px 0px 5px red;

text-shadow: -2px -2px 2px blue;

text-shadow:0px 0px 5px red;

text-shadow: 2px 2px 2px green;

text-shadow: 2px 2px 2px green;

text-shadow:-4px 4px 0px gainsboro;

text-shadow:-4px 4px 0px gainsboro;

text-shadow:-2px 2px 2px red, -4px 4px 9px gray;

text-shadow:-2px 2px 2px red, -4px 4px 9px gray;

text-shadow: -2px 2px 2px red, -4px 4px 9px gold;

text-shadow: -2px 2px 2px red, -4px 4px 9px gold;

text-shadow:-2px 2px 2px #fff, -3px 3px 2px #111, -5px 5px 2px #ccc;

text-shadow:-2px 2px 2px #fff, -3px 3px 2px #111, -5px 5px 2px #ccc;

Text Shadow Browsers Compatibility

Chrome, Safari, Firefox 4, Opera, and IE10 support Text Shadow without any vendor prefix (exp. text-shadow). But we need to add another property like -moz-text-shadow for Firefox 3.6 and earlier. IE 9 and below dowen't support text shadow
