Html5 web Storage

HTML5 Web storage includes local storage and session storage. Both can store data on client side, same like cookies.

Type of web storage

1. Local Storage


HTML5 Local Storage can store 5-10 mb data on client side same as cookies. Local storage can store data in key-value pair. Local storage data is easy to parse than cookies.

Add Local Storage

To add local storage, localstorage.setItem() method is used with two parameters. First parameter is name, and second method is value. We can also add local storage using localstorage.key=value;

<script>
localStorage.name="shifa";
localStorage.setItem("location","Noida"); localStorage.setItem("pincode",201301); </script>

Check Local Storage

To check local storage, use localStorage.getItem() method with one parameter Or localStorage.key

<script>
localStorage.name;
localStorage.getItem("location")
</script>

Remove Local Storage

To remove localstorage, use localStorage.removeItem() method is used with one parameter.

<script>
localStorage.removeItem("name");
localStorage.removeItem("location");
</script>

clear()

To remove all storage data

<script> localStorage.clear(); </script>

2. Session Storage


Session Storage can store data on client side, but only for current session. This means, once you close your web browser tab, all session storage data will be erased. Even for new tab in browser, session storage data is not accessible. Session storage data is also stored in key-valuepair. Refreshing or reloading browser tab cannot remove session data.

Add Session Storage

To add session storage, sessionStorage.setItem() method is used with two parameters. First parameter is name, and second method is value. We can also add session storage using sessionStorage.key=value;

<script>
sessionStorage.name="shifa"; sessionStorage.setItem("location","Noida");
sessionStorage.setItem("pincode",201301);
</script>

Check Session Storage

To check session storage, use sessionStorage.getItem() method with one parameter. Or sessionStorage.key

<script>
sessionStorage.name;
sessionStorage.getItem("location");
sessionStorage.getItem("pincode");
</script>

Remove Session Storage

To session localstorage, use sessionStorage.removeItem() method is used with one parameter


<script>
sessionStorage.removeItem("name");
sessionStorage.removeItem("location");
</script>

clear()

<script>
sessionStorage.clear();
</script>
