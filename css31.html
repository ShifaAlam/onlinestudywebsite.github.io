CSS3

CSS3 is the 3rd level and latest version of CSS, the stylesheet language of web used to control the styling of a website. CSS3 includes CSS 1, CSS 2, CSS 2.1 ( previous version of css) and CSS3 specifications .

CSS3 New Features Uses
Shadows Effect
Rounded Corners Easily round the corners of images and block elements
Gradients Linear, Radial and Repeating gradients for backgrounds.
Opacity Reduce opacity of Elements and Background colors using rbga colors.
Transitions Hover and Focus element with time and timing functions
Transformations Rotate, Scale, Skew and Translate any object
Animations Rotate, Scale, Skew and Translate any object
Multi Colunn layout Create multiple columns layouts.
Flexbox Flexbox is used to build one dimensional or 1d layout is CSS3.
Grids CSS3 Grids are used to build two dimensional or 2d layout is CSS3
@font face Embedded various fonts family.
@media Condition based CSS to create Responsive Web Design.
